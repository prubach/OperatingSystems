# Code examples used for the Operating Systems course at the Warsaw School of Economics.

1. monitor — Java example that demonstrates the use of monitors to synchronize threads.
2. buffer — Java example that demonstrates the use of a bounded buffer to synchronize threads.
3. semaphore_demo.c — C example that demonstrates the use of semaphores.
4. process_demo.c — C example that shows how to create multiple processes.
5. thread_demo.c — C example that shows how to create a single process but with many threads.

To use C examples, compile them with gcc and then run:

``gcc -o process_demo process_demo.c``
``./process_demo``
